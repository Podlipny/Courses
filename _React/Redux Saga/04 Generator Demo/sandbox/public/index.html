Redux Saga Sandbox

(F12 to open Developer Tools)

Console Reference

The following variables are available in the console (F12).

Property Name Description Sample Usage
effects All Redux Saga effects effects.call(parseFloat,"32a")
run Runs a generator as a saga run(function*(){yield effects.take("SET_VALUE")})
delay Delay Utility (from Redux Saga) yield delay(42);
dispatch Dispatches an action to the store in which sagas are running dispatch({type:"SET_VALUE",value:42})
co The co.js library co.wrap(function*(){return 42});
eventChannel The event channel constructor from Redux Saga const chan = yield eventChannel(()=>()=>{})
actionChannel The action channel constructor from Redux Saga const chan = yield actionChannel("MY_ACTION");
channel Generic channel constructor const chan = yield channel();
